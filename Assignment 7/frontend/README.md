# Student Feedback Review System - Frontend

This is the React frontend for the Student Feedback Review System.

## Getting Started

For complete setup instructions, development guide, and project overview, please refer to the main [README.md](../README.md) in the project root.

## Quick Start

```bash
npm install
npm run dev
```

The frontend will be available at `http://localhost:5173`

## Tech Stack

- React 19
- Vite
- React Router DOM
- Chart.js
- Lucide React
- JWT Decode
